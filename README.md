REQUIREMENTS 
Pillow is installed
    
HOW TO PLAY
python gui_blackjack.py

The game starts with a fixed initial balance and immediately deals the first hand.

Click the "HIT" button to request another card.

Click the "STAND" button to end your turn and initiate the Dealer's play.

After the hand is resolved, click the "DEAL" button to start a new round (assuming you have funds).

Known errors
Spamming "Hit" Can trigger an error where player hand value goes above 21 and breaks the game
